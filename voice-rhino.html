Voice-Aided Rhino Modeling

Speech or text becomes Rhino script through ASR, RAG, LLM generation, and plugin execution

StatuseCAADe 2025 相关成果
StackASR / RAG / LLM
Contribution参与系统流程、语音/文本交互逻辑、Rhino 执行链路与案例测试表达
01 Problem

建筑师需要更低门槛地把口头意图转化为 Rhino 操作与脚本。

02 Method

ASR + RAG + LLM 生成脚本，再通过插件执行并记录反馈。

03 Output

覆盖创建、修改、删除和复杂建模的语音辅助原型链路。

04 Contribution

主要参与交互流程、系统链路与测试案例表达。

Problem

项目问题

建筑建模命令复杂、脚本门槛高，用户希望通过语音或文本快速完成创建、修改、删除和复杂建模任务。

Method

方法流程

  1. 构建 ASR、RAG、LLM 脚本生成、Rhino 插件执行和反馈记录的系统架构。
  2. 用 Rhino 错误日志与 API 示例为 LLM 提供针对性修复证据，形成闭环错误修复机制。
  3. 保留多轮脚本历史，使后续 prompt 能基于前文进行迭代建模。

Output

成果价值

50 条指令试验显示基础与修改任务成功率强，复杂任务自动化耗时相对手工有明显下降，主要瓶颈集中在响应速度。
