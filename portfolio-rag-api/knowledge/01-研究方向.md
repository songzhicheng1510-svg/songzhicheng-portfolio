# 01-研究方向

## 总体研究方向

宋志诚的研究方向可以概括为 AI 建筑设计工作流、AIGC 空间可视化、Rhino 自动化建模和城市更新多智能体系统。

这些方向都围绕一个核心问题：如何让 AI 不只是生成图片或文字，而是进入建筑设计的真实流程，支持方案生成、空间更新、建模执行、反馈记录和可解释表达。

## 方向一：AI-assisted architectural design workflows

该方向关注建筑设计早期阶段中，自然语言任务书、规范约束、设计策略和评估指标如何转化为可运行、可复查、可比较的工作流。

代表案例是 Text to Massing。该项目将建筑 brief、设计约束和指标评估组织成 Prompt Pack，并通过 LLM 生成 Rhino Python 体量脚本。

## 方向二：AIGC 空间可视化

该方向关注图像、全景图、街景视频和建筑分析图如何通过 AIGC 工具进入空间表达与沟通流程。

代表案例包括 EditPanorama、AI-Driven Street-View Video Generation 和 AI-Assisted Architectural Image Analysis & Generation。

## 方向三：Rhino 自动化建模与 AI 工具原型

该方向关注如何降低建筑师使用 Rhino 脚本和复杂建模命令的门槛，使语音、文本、错误日志和 API 示例能够共同支持脚本生成、执行与修复。

代表案例是 Voice-Aided Rhino Modeling。该项目将 ASR、RAG、LLM 和 Rhino 插件执行链路组合起来，把用户的语音或文本意图转化为 Rhino 建模操作。

## 方向四：城市更新多智能体系统

该方向关注城市微空间更新中调研周期长、跨专业反馈难以闭环的问题，尝试用虚拟居民、专家代理、地图数据、天气和交通信息构建自动诊断与生成式策略流程。

代表案例是 Building Multimodal AI Agents for Urban Renewal。

## 研究方法特征

- 从真实建筑问题出发，而不是只演示 AI 效果。
- 强调输入、处理、输出的工作流结构。
- 注重可复现性，例如脚本、约束、评估指标和流程图。
- 用角色、状态、工具栈和贡献描述团队项目，避免把协作成果包装成个人独立完成。
- 同时面向学术申请、实习岗位和产品型机会。
